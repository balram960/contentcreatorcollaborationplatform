# Content Creator Collaboration Platform

A modern and responsive Content Creator Collaboration Platform built with React, TypeScript, Vite, Tailwind CSS, and Supabase. The platform enables creators to discover collaboration opportunities, manage events, register for activities, and securely authenticate using email-based Sign In and Sign Up.

## ✨ Features

- 🔐 User Authentication (Sign Up & Sign In)
- 👤 Secure user account management
- 🏠 Responsive Home Dashboard
- 📅 Browse available collaboration events
- 📖 View detailed event information
- ✅ Register for events
- 📋 My Registrations page
- 🛠️ Admin Panel for event management
- ⚡ Fast and responsive UI
- 📱 Mobile-friendly design
- ☁️ Supabase backend integration

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/balram960/contentcreatorcollaborationplatform.git
```

### Navigate to Project

```bash
cd contentcreatorcollaborationplatform
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the project root.

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

---

### Start Development Server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
